# Codex Summary — Mahjong HTML5

- Добавлена папка `mahjong/` с полноценной HTML5-реализацией маджонга (Mahjong Solitaire):
  - `index.html` — разметка приложения с панелью статистики и кнопками управления.
  - `styles.css` — визуальное оформление, адаптивные стили и состояния тайлов.
  - `game.js` — игровая логика: генерация раскладки, выбор пар свободных тайлов, подсказки, перемешивание оставшихся фишек, таймер и счётчик ходов.
- Проверено исполнение JS через `node --check mahjong/game.js`.
- Чтобы поиграть, открой `mahjong/index.html` в браузере (локально без сервера).
